# DAX DEPO – Power BI Project

## 📊 Project Overview
DAX DEPO is a Power BI dashboard project focused on advanced DAX calculations, sales analysis, and business intelligence reporting.  
This project demonstrates the use of Power BI data modeling, calculated columns, measures, filters, and interactive dashboards.

---

## 🚀 Features
- Interactive Power BI Dashboard
- Advanced DAX Measures
- Sales Performance Analysis
- Profit & Loss Tracking
- Region-wise Data Insights
- KPI Cards and Visualizations
- Dynamic Filtering and Slicers

---

## 🛠 Tools & Technologies
- Microsoft Power BI
- DAX (Data Analysis Expressions)
- Data Modeling
- Power Query
- Excel / CSV Dataset Integration

---

## 📁 Project File
- `Dax Depo Project.pbix`

---

## 📌 Sample DAX Query

```DAX
High Value Transactions =
COUNTX(
    FILTER(Sales_Fact, Sales_Fact[Sales Amount] > 1000),
    Sales_Fact[SalesID]
)

```
